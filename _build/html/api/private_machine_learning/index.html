private_machine_learning¶

private_SGD

This Differencial Privacy(DP) SGD proposed in https://arxiv.org/pdf/1607.00133.pdf.
private_SGD(gradient_function, get_weights_function, update_weights_function, learning_rate_function, train_data, group_size, gradient_norm_bound, number_of_steps, sigma, moment_privacy_budget_tracker, test_interval=None, test_function=None)[source]¶

This Differencial Privacy(DP) SGD proposed in https://arxiv.org/pdf/1607.00133.pdf. This privacy budget is calculated using MomentPrivacyBudgetTracker.

Parameters

  • gradient_function (Callable[[], Any]) – Function that receives data batch as input and return the gradient of the training model.

  • get_weights_function (Callable[[], None]) – Function that return the weight of the training model.

  • update_weights_function (Callable[[Any], None]) – Function that receives new weights as input and update the weight of the model.

  • learning_rate_function (Callable[[int], float]) – Fucntion that receives step size as input and return the learning rate.

  • train_data (array) – Array of traning data. Each element of the array is a training sample.

  • group_size (int) – Number of data used for training in one step.

  • gradient_norm_bound (Union[int, float]) – L2-norm bound of the gradient.

  • number_of_steps (int) – Number of training steps.

  • sigma (Union[int, float]) – Value of sigma. Greater sigma correspond to higher noise and lower privacy budget.

  • moment_privacy_budget_tracker (MomentPrivacyBudgetTracker) – Instance of MomentPrivacyBudgetTracker.

  • test_interval (Optional[int]) – test_function will be triggred every test_interval steps if test_function is specify , defaults to None

  • test_function (Optional[Callable[[], None]]) – Fucntion to test the performance of model, defaults to None
