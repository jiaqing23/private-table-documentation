exponential_mechanism(x, score_function, sensitivity, privacy_budget)[source]¶

Differentially private exponantial mechanism. Each keys sampling by probability proportional to:

\[\exp\left(\frac{\epsilon \times score}{2 \Delta f}\right)\]

The result guarantees \((\epsilon,\delta)\)-differential privacy.

Parameters

  • x (ndarray) – Sensitive input data

  • score_function (Callable[[ndarray], ndarray]) – a function to receive x and return a dictionary with items {element: score}

  • sensitivity (float) – The global L1-sensitivity \(\Delta f\) of x

  • privacy_budget (PrivacyBudget) – The privacy budget \((\epsilon,0)\) used for the outputs

Return type

Any

Returns

The sampled element

gaussian_mechanism(x, sensitivity, privacy_budget)[source]¶

Differentially private Gaussian mechanism. Add Gaussian noise to the value:

\[x + Normal\left(\mu=0, \sigma=\frac{\sqrt{2\log(1.25/\delta)}\Delta f}{\epsilon} \right)\]

The result guarantees \((\epsilon,\delta)\)-differential privacy.

Parameters

  • x (Union[int, float, ndarray]) – Sensitive input data

  • sensitivity (float) – The global L2-sensitivity \(\Delta f\) of x

  • privacy_budget (PrivacyBudget) – The privacy budget \((\epsilon,\delta)\) used for the outputs

Return type

Union[float, ndarray]

Returns

Input data protected by noise
